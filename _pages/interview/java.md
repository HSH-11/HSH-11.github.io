# Java 면접 질문 정리

여기에 Java 관련 면접 질문과 답변을 정리해주세요.

## JVM

### JVM이란?
JVM(Java Virtual Machine)은 Java 바이트코드를 실행하는 가상 머신입니다.

### JVM의 역할은?
- 바이트코드 로딩
- 메모리 관리
- 가비지 컬렉션

## 가비지 컬렉션

### 가비지 컬렉션이란?
사용하지 않는 객체를 자동으로 메모리에서 제거하는 프로세스입니다.

### GC 알고리즘 종류는?
- Serial GC
- Parallel GC
- G1 GC
- ZGC
